# The Lost Code of Algoria

Game story petualangan teks yang berlatar dunia digital Algoria.

## Deskripsi
Anda adalah seorang programmer yang terjebak di dunia digital Algoria. 
Untuk kembali ke dunia nyata, Anda harus menemukan "The Lost Code" yang 
tersembunyi di Server Core. Hadapi Bug Monster, pecahkan puzzle kode, 
dan kumpulkan pengetahuan untuk menyelesaikan petualangan ini!

## Fitur
- Game story berbasis teks dengan narasi interaktif
- Sistem inventory dan penggunaan item
- Pertempuran melawan musuh
- Puzzle untuk dipecahkan
- Multiple lokasi untuk dijelajahi
- Sistem knowledge/pengetahuan
- Interface berwarna dengan Colorama
